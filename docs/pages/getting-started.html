Getting Started

Everything you need to start building beautiful interfaces with the ajaRed Design System.

Installation

1. Via CDN (Quickest)

Add the following to your HTML <head> tag:

<!-- ajaRed Design System CSS --> <link rel="stylesheet" href="https://unpkg.com/@ajared/design-system/dist/ajared.min.css">

2. Via npm

Install the package using npm or yarn:

# Using npm npm install @ajared/design-system # Using yarn yarn add @ajared/design-system

Then import in your project:

// In your main CSS file @import '@ajared/design-system/dist/ajared.css'; // Or in your JavaScript import '@ajared/design-system/dist/ajared.css';

3. Download Directly

Download the latest release from our GitHub repository and include the CSS file in your project:

<link rel="stylesheet" href="path/to/ajared.min.css">

Quick Start

Once you've included the CSS, you can start using components immediately. Here's a simple example:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>My ajaRed App</title> <link rel="stylesheet" href="https://unpkg.com/@ajared/design-system/dist/ajared.min.css"> </head> <body> <div style="padding: 2rem;"> <h1 class="ajr-h1">Hello, ajaRed!</h1> <p class="ajr-body">Get started with beautiful components.</p> <button class="ajr-button ajr-button--primary ajr-button--md"> Get Started </button> </div> </body> </html>

💡 Tip: All ajaRed components use the ajr- prefix to avoid conflicts with your existing styles.

Using Components

Buttons

Create buttons with different variants and sizes:

<!-- Primary button --> <button class="ajr-button ajr-button--primary ajr-button--md">Primary</button> <!-- Secondary button --> <button class="ajr-button ajr-button--secondary ajr-button--md">Secondary</button> <!-- Small tertiary button --> <button class="ajr-button ajr-button--tertiary ajr-button--sm">Tertiary</button>

Form Inputs

Build accessible forms with validation states:

<div class="ajr-input-group"> <label class="ajr-input-label">Email Address</label> <input type="email" class="ajr-input" placeholder="your@email.com"> <span class="ajr-input-helper">We'll never share your email.</span> </div> <!-- Input with error state --> <input type="text" class="ajr-input ajr-input--error" placeholder="Invalid input">

Cards

Organize content with flexible card layouts:

<div class="ajr-card"> <div class="ajr-card-header"> <h3>Card Title</h3> </div> <div class="ajr-card-body"> <p>Card content goes here.</p> </div> <div class="ajr-card-footer"> <button class="ajr-button ajr-button--primary ajr-button--sm">Action</button> </div> </div>

Alerts

Display feedback messages with different severity levels:

<!-- Success alert --> <div class="ajr-alert ajr-alert--success"> <div class="ajr-alert-content"> <strong>Success!</strong> Your changes have been saved. </div> </div> <!-- Error alert --> <div class="ajr-alert ajr-alert--error ajr-alert--outlined"> <div class="ajr-alert-content"> <strong>Error!</strong> Something went wrong. </div> <button class="ajr-alert-close">×</button> </div>

Using Design Tokens

All design tokens are available as CSS custom properties. Use them in your custom components for consistency:

.my-custom-component { /* Colors */ background-color: var(--color-primary); color: var(--color-text-inverse); /* Spacing */ padding: var(--space-4) var(--space-6); margin-bottom: var(--space-8); /* Typography */ font-family: var(--font-primary); font-size: var(--font-size-lg); font-weight: var(--font-weight-semibold); /* Border & Shadow */ border-radius: var(--radius-md); box-shadow: var(--shadow-md); /* Transitions */ transition: transform var(--transition-fast); }

View all available tokens on the Design Tokens page.

Framework Integration

React

import '@ajared/design-system/dist/ajared.css'; function App() { return ( <div> <h1 className="ajr-h1">Hello React!</h1> <button className="ajr-button ajr-button--primary ajr-button--md"> Click Me </button> </div> ); }

Vue

// In main.js or App.vue import '@ajared/design-system/dist/ajared.css'; // In your component <template> <div> <h1 class="ajr-h1">Hello Vue!</h1> <button class="ajr-button ajr-button--primary ajr-button--md"> Click Me </button> </div> </template>

Vanilla JavaScript

// Just include the CSS in your HTML <link rel="stylesheet" href="path/to/ajared.css"> // Use components directly const button = document.createElement('button'); button.className = 'ajr-button ajr-button--primary ajr-button--md'; button.textContent = 'Click Me'; document.body.appendChild(button);

Accessibility

All ajaRed components are built with accessibility in mind and follow WCAG 2.1 AA guidelines:

  • Keyboard Navigation: All interactive components support keyboard navigation
  • Screen Readers: Proper ARIA labels and roles are included
  • Color Contrast: All text meets minimum contrast ratios (4.5:1)
  • Focus States: Clear visual focus indicators for keyboard users
  • Semantic HTML: Components use proper semantic elements

♿ Accessibility Note: Always add appropriate ARIA labels and roles for dynamic content and ensure proper heading hierarchy in your pages.

Browser Support

ajaRed Design System supports all modern browsers:

  • Chrome (latest 2 versions)
  • Firefox (latest 2 versions)
  • Safari (latest 2 versions)
  • Edge (latest 2 versions)

Internet Explorer is not supported as we use modern CSS features like custom properties and flexbox/grid.

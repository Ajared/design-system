MOTION · TIMINGS + EASING
Motion at Ajared is restrained. Most things should feel like a page turning, not a slot machine. Use snap for direct-manipulation; standard for navigational transitions; spring only for welcoming moments.
Snap
180ms · cubic-bezier(0.4, 0, 0.2, 1)
Standard
900ms · cubic-bezier(0.2, 0, 0, 1)
Spring (welcome)
1200ms · cubic-bezier(0.34, 1.56, 0.64, 1)
Fade (ambient)
1400ms · ease-in-out · loop
Moiré drift
8s · linear · infinite · backgrounds only
Reduce-motion: substitute all animations with 200ms opacity cross-fades. Honor prefers-reduced-motion.
